This is a collection of setting files for each airport parking lot of FSDreamTeam GSX.

Requirements

  • FSDreamTeam - GSX Ground Services X 2.0 later
  • FSX or P3D with installed GSX

Recording status in this release

Those whose recording status is in RED are the settings that were new / updated in this release.

File nameAirport Name(ICAO code)Name for Add-onStatus
rjaa-wci.ini Narita Intl(RJAA) WingCreation RJAA rev.1 New (Only Terminal 1)
rjnt-fssa.ini Toyama(RJNT) FS Scenery Atelier RJNT rev.1 New
rjns-wci.ini Shizuoka(RJNS) WingCreation RJNS rev.1 New
rjfm-fssa.ini Miyazaki(RJFM) FS Scenery Atelier RJFM rev.1 New

Each file works effectively only at the add-on airport listed in Name for Add-on.
Please check the recording situation against the add-on scenery installed in your flight sim environment.
(The leftmost checkbox is for confirmation for you. Please use it freely.)

Installation

  1. Move to the folder at %AppData%\Roaming\Virtuali\GSX\.
  2. If you have an old configuration file for the airport you wish to install in the future, please backup and delete.
  3. Move to ini file here.
  4. Open the ini file and add AFCAD BGL file path into the afcad_path = .
